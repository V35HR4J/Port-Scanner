# 🕵🏻‍♀️Port-Scanner 🕵🏻‍♀️
A Simple Port Scanner By newbie 👷

# 🤸 Usage 🤸
U can scan host by specifying starting and ending ports:<br><br>
`python3 scanner.py {ip} {startingPort} {endingPort}`<br><br>
         or simply,<br><br>
`python3 scanner.py {ip}` only
